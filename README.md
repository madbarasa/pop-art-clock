# Pop Art Clock ⚡

一个充满活力的 **Pop Art 风格** 网页闹钟，功能完整、视觉大胆！

## 🎨 特点

- **大字号实时时钟** - 清晰醒目，带日期显示
- **闹钟管理** - 添加/删除/开关闹钟，支持自定义标签
- **计时器** - 设置时/分/秒，开始/暂停/重置
- **Pop Art 视觉** - 大胆配色、粗黑边框、硬阴影效果
- **完全离线** - 纯静态 HTML，无需服务器

## 🚀 在线访问

https://madbarasa.github.io/pop-art-clock/

## 📱 使用说明

1. 打开网页即可使用
2. 时钟每秒自动更新
3. 闹钟到点会弹窗提醒 + 提示音
4. 数据保存在浏览器本地（刷新不丢失）

## 🔧 本地运行

```bash
# 直接用浏览器打开 index.html 即可
open index.html
```

## 📝 技术栈

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript
- LocalStorage (数据持久化)
- Web Audio API (闹钟提示音)
